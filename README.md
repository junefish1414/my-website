## 專案說明
* 網站主要使用 `Vue.js` 和 `Vue Cli` 架設而成
* 引入 `Aos Master` 做滾動框架動畫
* 使用 `Component `元件將區塊分開處理
* 使用 `SCSS` 預處理器統整風格與色系
* 加入 `ESLint Standard `風格控管程式碼品質
* 使用 `Adobe Xd` 設計與 `Export`
